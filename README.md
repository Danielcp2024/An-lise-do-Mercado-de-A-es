# Análise do Mercado de Ações – Google Sheets
## Descrição do Projeto

Projeto de análise exploratória de dados (EDA) realizado no Google Sheets a partir de registros de operações do mercado de ações. O objetivo foi organizar, tratar e analisar dados financeiros para identificar padrões, tendências e comportamentos relevantes para suporte à tomada de decisão.

## Ferramentas Utilizadas

Google Sheets

## Atividades Realizadas

Limpeza e padronização de dados

Organização temporal (dia, mês e ano)

## Análise por tipo de operação (compra e venda)

Aplicação de filtros, fórmulas e ordenações

Análise exploratória de preços e ativos

Identificação de padrões de comportamento do mercado

## Resultados

O projeto permitiu estruturar uma base de dados confiável, facilitando a visualização das informações e a extração de insights sobre movimentações do mercado de ações.

## Objetivo Profissional

Demonstrar habilidades em análise de dados, organização de informações e uso prático do Google Sheets aplicado a cenários financeiros.
